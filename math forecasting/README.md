# Модели математического прогнозирования: Holt–Winters, Линейная регрессия, ARIMA

Данный репозиторий содержит три самостоятельных работы, выполненных в рамках курса по математическому прогнозированию.  
Каждая работа представляет собой исследование и практическую реализацию моделей анализа временных рядов с последующей оценкой качества прогнозов.

---

# Работа 1. Модель Хольта–Уинтерса (Holt–Winters)

**Файл:** `holt_winters_model.ipynb`  
**Статус:** завершено

## Цель  
Реализовать и исследовать модель Хольта–Уинтерса, выполнить тюнинг параметров сглаживания и (опционально) длины сезона, построить прогноз временного ряда.

## Основные задачи  
- Реализация модели Holt–Winters (trend + seasonality).  
- Ввод пользовательских параметров:  
  - коэффициенты сглаживания,  
  - длина сезона,  
  - горизонт прогноза (1–2 периода).  
- Вывод параметров модели: тренд, сезонность, прогноз.  
- Режим тюнинга: подбор параметров сглаживания и длины сезона.  


## Результаты  
- Реализован полный workflow модели Хольта–Уинтерса.  
- Выполнен подбор оптимальных параметров.  
- Получены прогнозы на несколько шагов вперёд.

# Работа 2. Линейная многофакторная регрессия (ЛМФМ)

**Файл:** `Linear_regression.ipynb`  
**Статус:** завершено

## Цель  
Построить, оценить и оптимизировать линейную многофакторную модель (ЛМФМ) для временного ряда с факторами.

## Основные задачи  
- Оценка параметров линейной регрессии.  
- Многоэтапный отбор факторов:  
  - по статистической значимости,  
  - по корреляции между факторами,  
  - по связи факторов с откликом.  
- Проверка адекватности модели:  
  - значимость коэффициента детерминации (F-тест),  
  - вычисление ошибок (RMSE или альтернативные метрики),  
  - вывод показателей качества.  
- Предсказание по новой комбинации факторов.  
- Возможность применения:  
  - временных лагов для факторов,  
  - выделения подотрезков отклика.  
- Тюнинг параметров для минимизации ошибки.

## Результаты  
- Построена и оценена ЛМФМ;  
- выполнен отбор факторов;  
- проверена адекватность модели;  
- выполнено прогнозирование;  
- модель оптимизирована по ошибке и F-статистике.

# Работа 3. Модели ARIMA и SARIMA

**Файл:** `SARIMA.ipynb`  
**Статус:** завершено

## Цель  
Построить прогноз сезонного временного ряда с использованием моделей ARIMA/SARIMA, выполнить декомпозицию, сравнить прогнозы с моделью Хольта–Уинтерса.

## Основные задачи  
1. Выбор временного ряда с выраженной сезонностью.  
2. Проведение сезонной декомпозиции (`seasonal_decompose`).  
3. Анализ временного ряда: ACF, PACF.  
4. Разделение данных на обучающую и тестовую выборки.  
5. Обучение моделей ARIMA / SARIMA.  
6. Визуализация:  
   - исходный ряд,  
   - тренд/сезонность/остаток,  
   - прогноз vs факт на тесте.  
7. Проверка адекватности моделей.  
8. Сравнение с моделью Holt–Winters.  
9. Тюнинг параметров ARIMA и оценка улучшения метрик.

## Результаты  
- Проведён анализ сезонности и структурных компонент.  
- Построены модели ARIMA/SARIMA и выполнен тестовый прогноз.  
- Качество подтверждено визуально и метриками.  
- Выполнено сравнение с моделью Хольта–Уинтерса.  
- При тюнинге параметры улучшают точность прогноза.
