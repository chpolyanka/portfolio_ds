# Исследование факторов, влияющих на размер инвестиционного портфеля клиентов банка

**Статус:** завершён

## Цель
Определить, какие демографические, поведенческие и финансовые признаки отличают клиентов с крупным инвестиционным портфелем. 

Датасет: https://dano.hse.ru/mirror/pubs/share/987943358.rar

## Описание
Проведён EDA клиентской базы банка:
- обработаны пропуски и некорректные данные,
- выделен таргет — крупный инвестиционный портфель,
- проанализированы демография, активность и финансовое поведение,
- сравнение групп выполнено по медианам, квантилям и распределениям,
- построены ключевые визуализации.

## Основные выводы
- Топ-клиенты старше и имеют больший стаж.  
- Выше активность (особенно чтения и вовлечённость).  
- Финансовые различия заметнее в верхних квантилях.  
- Объём и структура портфеля — ключевые признаки сегментации.

## Инструменты
Python, Pandas, NumPy, Matplotlib, Seaborn

## Навыки
EDA, обработка пропусков, визуализация, работа с признаками, клиентская аналитика

## Ключевые слова
EDA, банковская аналитика, портфель клиента, предобработка данных, визуализация
