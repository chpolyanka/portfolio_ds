# Multivariate Time Series Forecasting: ARIMA & VAR Models

**Статус:** завершён  
**Цель:** изучить возможности одно- и многомерных моделей временных рядов (ARIMA/SARIMA и VAR) для прогноза макроэкономических показателей.

---

## Описание проекта
В работе проведён анализ нескольких экономических временных рядов:

- уровень инфляции,  
- курс доллара к рублю,  
- средняя цена нефти за месяц.

В проекте реализованы и сравниваются два подхода:

### 1. (S)ARIMA  
- анализ стационарности;  
- подбор параметров модели;  
- построение прогноза;  
- оценка качества с помощью RMSE.

### 2. VAR (Vector AutoRegression)  
- анализ взаимосвязи нескольких временных рядов;  
- подбор лага модели;  
- тесты причинности (Granger causality);  
- построение мультивариантного прогноза.

---

## Основные выводы

- Добавление дополнительных экономических переменных в VAR улучшает качество прогноза по сравнению с классической ARIMA.  
- На лаге 2 статистические тесты дают слабые признаки причинности между макропоказателями.  
- RMSE прогнозов снижается при использовании информации о средней цене нефти.  
- SARIMA и VAR способны уловить сезонность и тренды, но VAR более информативна для связанных экономических показателей.
