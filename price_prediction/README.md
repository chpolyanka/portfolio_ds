# Real Estate Price Prediction: Linear Regression, XGBoost и Квантильная регрессия

**Статус:** завершён  
**Данные:** открытый датасет стоимости квартир в Москве

---

## Цель проекта
Сравнить эффективность разных подходов к прогнозированию цен на жильё:  
- линейной регрессии,  
- модели градиентного бустинга XGBoost,  
- квантильной регрессии (Quantile Regression).

Цель — понять, какие модели лучше предсказывают стоимость объектов с разными характеристиками и распределением цен.

---

## Описание проекта

В ноутбуке выполнены следующие шаги:

### 1. Линейная регрессия (LinearRegression)
- Модель плохо экстраполирует значения на дорогом сегменте.
- Предполагает линейность и равномерность распределений, что не соответствует рынку недвижимости.

### 2. XGBoost Regressor
- Лучшая производительность среди всех моделей.
- Учитывает нелинейные зависимости и сложные признаки.
- Ошибки возникают из-за отсутствия признаков премиальности (окружение, инфраструктура).

### 3. Quantile Regression
- Построена квантильная модель (0.5, 0.9 и др.).
- Модель позволяет оценивать не только среднее, но и распределение возможных цен.
- Ошибки высоких квантилей связаны с недостатком данных о премиальном жилье.

### 4. Оценка качества
- Выполнено сравнение моделей по метрикам.
- XGBoost показал наилучшие результаты.
- Линейная модель недооценивает дорогие объекты.
- Квантильная регрессия полезна для оценки диапазона цен, но требует более информативных признаков.

---

## Основные выводы

- **XGBoost — самая точная модель** для прогнозирования цен.  
- **LinearRegression** сильно упрощает динамику рынка и плохо работает на дорогом сегменте.  
- **Quantile Regression** даёт более глубокое понимание распределения стоимости, но хуже справляется без богатого набора признаков.  
- Реальный рынок недвижимости требует моделей, способных улавливать нелинейность, выбросы и асимметрию распределений.
