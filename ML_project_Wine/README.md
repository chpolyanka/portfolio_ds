# Анализ качества вина и применение методов машинного обучения

**Статус:** завершён

## Цель работы
Использовать методы анализа данных и машинного обучения для изучения химических характеристик вин и их связи с качеством. Выполнить задачи по прогнозированию, кластеризации и визуализации данных.

## Задачи проекта
1. Изучение структуры данных и предварительная обработка.  
2. Визуализация влияния химических характеристик на качество.  
3. Классификация вина по качеству.  
4. Прогнозирование рейтинга вина (регрессия).  
5. Кластеризация вин по химическим характеристикам.  
6. Оптимизация «рецепта» вина для заданного качества.  
7. Проведение экспериментов для оценки моделей.  
8. Создание веб-интерфейса для модели предсказания качества вина.

## Описание проекта
Исследованы химические показатели красных и белых вин (кислотность, сахар, pH, плотность, SO₂, алкоголь и др.) с целью выявить их влияние на качество.  
В проекте выполнены:

- EDA и визуализация распределений;  
- классификация качества вина (RandomForest, GradientBoosting);  
- регрессия для прогноза рейтинга;  
- кластеризация KMeans;  
- анализ важности признаков;  
- оптимизация характеристик вина с помощью алгоритмов SciPy;  
- разработка веб-интерфейса, позволяющего:
  - изменять характеристики вина с помощью интерактивных слайдеров,
  - получать прогноз качества,
  - визуализировать влияние признаков (бар-чарт SHAP-значений).

## Основные результаты
- Модели успешно прогнозируют качество и выявляют значимые химические параметры.  
- Алкоголь, сернистый ангидрид, сульфаты и плотность — ключевые факторы качества.  
- Кластеризация выделяет естественные группы вин.  
- Реализованный веб-интерфейс позволяет удобно исследовать влияние параметров на предсказание и изучать работу модели.

## Инструменты
Python, Pandas, NumPy, Matplotlib, Seaborn, scikit-learn (RandomForest, GradientBoosting, KMeans, GridSearchCV), SciPy, Streamlit (или аналог) для веб-интерфейса.

## Навыки
EDA, визуализация, классификация, регрессия, кластеризация, подбор гиперпараметров, оптимизация, интерпретация моделей, прототипирование веб-интерфейсов для ML.
